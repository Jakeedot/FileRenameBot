## Rename Bot 

An Open Source Telegram Rename Bot

---
Rename any Telegram Files with Permanent Thumbnail Support

* Added auto detect of files


### Deploy on Heroku

[![Deploy](https://www.herokucdn.com/deploy/button.svg)](https://heroku.com/deploy?template=https://github.com/Jakeedot/Filerenamebot)

Required variables
* `TG_BOT_TOKEN` Bot token from @BotFather
* `API_ID` Value from my.telegram.org
* `API_HASH` Value from my.telegram.org
* `AUTH_USERS` add user's id only for preferred users to use else put 0
* `BANNED_USER` add user's id if want to block them else put 0
